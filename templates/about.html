Project Objective

This machine learning system predicts gold price per gram in USD on a monthly basis using advanced time-series forecasting techniques.

ML Prediction

Gold price per gram in USD

Display

USD exchange rate equivalents

Features

CPI, Fed Funds, Real Rate, Treasury 10Y

Phase-wise Methodology

  • Load and inspect all CSV datasets
  • Standardize date formats and create year_month merge key
  • Handle different frequencies (monthly vs daily)
  • Convert daily data to monthly using monthly averages
  • Convert gold prices from USD/ounce to USD/gram
  • Handle missing values (forward fill, backward fill)
  • Merge all datasets on year_month (1975-2025)

  • Log Transformation: Gold prices converted to log scale for stationarity
  • Macro Features: CPI YoY Inflation, Real Interest Rate, Yield Spread
  • Lag Features: Gold (lag1, lag3), Fed Funds (lag1), CPI YoY (lag1), Real Rate (lag1)
  • Technical Features: Gold MA3, Gold return (1-month log return)
  • Target: log_gold_next (log of next month's gold price)
  • Note: Model predicts log scale to handle non-stationarity. Predictions are back-transformed to USD/gram for display.
